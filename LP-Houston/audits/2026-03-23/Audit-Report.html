Duplicate Conversion Actions — Brochure & Form Counted Twice

Download Brochure is tracked as BOTH a website conversion action (1/click) AND a GA4 event (many/click). Same for the contact/form flow. This means a single brochure download by one user can register 2+ conversions. The reported "42 conversions" is likely inflated — actual unique lead events are probably 25–35.

Brochure Downloads = Real Conversions

Brochure downloads require a contact form fill to access — they are qualified lead events, not bounce events. They should remain in the conversion set and NOT be demoted to "other conversion." The quality concern is duplication, not the action type itself.

Recommended Fix

Disable the older website-tag versions (LP Houston web — Contact Form, Download Brochure, Free Estimate) if they duplicate GA4 events. Keep ONE source per action. This will improve bidding signal quality and accurate lead reporting for Donavan's meeting.
